# 日程管理器（AI自动化脚本版）

## 功能介绍

这是一个基于 Tampermonkey 的日程管理脚本，集成了智谱AI自动化功能，可通过自然语言指令自动管理日程。

### 核心功能

- 📅 **多视图日历**：支持日、周、月三种视图，直观查看不同周期的日程安排
- 🤖 **AI 自动化**：通过智谱AI接口，使用自然语言指令管理日程
- 🎯 **日程操作**：添加、编辑、删除、标记完成、清空日程
- 🏷️ **标签系统**：支持重要且紧急、重要、紧急、一般四种标签
- 📋 **子任务支持**：可为日程添加子任务，更细致地管理任务
- 📤 **导入导出**：支持JSON格式的日程数据导入导出，方便备份和分享
- 🗑️ **回收站**：删除的日程会进入回收站，可查看、恢复或清空
- 💾 **本地存储**：使用 Tampermonkey 存储 API 持久化日程数据
- 📱 **界面交互**：支持拖拽移动、最小化、响应式设计
- 🌙 **深色主题**：现代化深色界面，减少视觉疲劳

## 安装方法

1. 安装 Tampermonkey 浏览器扩展
2. 复制 `日程管理器.js` 的完整代码
3. 在 Tampermonkey 中创建新脚本，粘贴代码并保存
4. 刷新浏览器，脚本将自动加载

## 使用说明

### 基本操作
1. **视图切换**：点击顶部的 "日"、"周"、"月" 按钮切换不同视图
2. **周期切换**：使用顶部的左右箭头切换不同周期
3. **添加日程**：点击底部 "+ 添加日程" 按钮，输入内容、时间和标签
4. **编辑日程**：点击底部 "编辑日程" 按钮，输入日程ID进行编辑
5. **导入日程**：点击底部 "导入" 按钮，粘贴JSON格式的日程数据
6. **导出日程**：点击底部 "导出" 按钮，将日程导出为JSON文件
7. **回收站**：点击底部 "回收站" 按钮，查看、恢复或清空回收站
8. **清空所有**：点击底部 "清空所有" 按钮，确认后将所有日程移至回收站
9. **拖拽移动**：拖动顶部蓝色标题栏可移动窗口
10. **最小化**：点击顶部 "-" 按钮最小化窗口
11. **关闭**：点击顶部 "×" 按钮关闭窗口

### AI 自动化操作
1. 在 "输入智谱API Key" 输入框中填写您的智谱AI API Key
2. 选择合适的 AI 模型（默认使用 GLM-4.7-Flash）
3. 在 "输入AI指令" 框中输入自然语言指令，例如：
   - "添加下午3点开会"
   - "删除所有会议"
   - "标记所有日程为完成"
   - "清空所有日程"
4. 点击 "执行AI" 按钮，系统会自动执行相应操作

## 核心逻辑说明

### 1. 界面初始化

- 创建固定定位的日历容器，默认为月视图
- 加载智谱AI API Key、模型选择、窗口位置等配置信息
- 实现拖拽、最小化、关闭等窗口控制功能
- 加载保存的配置和位置信息

### 2. 日历渲染

- **日视图**：显示当天的详细日程，包括标签和子任务
- **周视图**：显示一周的日程概览
- **月视图**：显示一个月的日程概览
- 计算当前周期的日期范围和起始星期
- 生成日历网格，包含日期单元格
- 显示每天的日程列表，区分已完成和未完成状态
- 高亮显示当天日期
- 点击指定日期高亮，并跳转到该日期的日程列表页

### 3. 日程管理

- **添加**：生成唯一ID，保存到本地存储，支持标签和子任务
- **编辑**：根据ID修改日程内容、时间、标签和子任务
- **删除**：根据ID或关键词过滤删除，删除的日程会进入回收站
- **完成**：标记单个或所有日程为已完成
- **清空**：将所有日程移至回收站
- **导入**：导入JSON格式日程数据，合并到本地存储
- **导出**：将当前日程导出为JSON文件，方便备份和分享
- **回收站**：查看、恢复或清空回收站中的日程

### 4. AI 集成

- 调用智谱AI API，发送系统提示和用户指令
- 解析AI返回的JSON格式指令
- 执行相应的日程操作，包括添加、删除、标记完成、清空和编辑
- 处理API请求状态和错误
- 支持通过AI添加带标签和子任务的日程

### 5. 数据存储

- 使用 Tampermonkey 的 GM_setValue 和 GM_getValue API
- 持久化存储日程数据、API Key、模型选择和窗口位置

## 技术实现

### 技术栈

- JavaScript (ES6+)
- Tampermonkey API
- Tailwind CSS (CDN)
- 智谱AI API

### 关键函数
- `createUI()`：创建用户界面
- `renderCalendar()`：渲染月历和日程
- `saveSchedules()`：保存日程数据
- `sendAIScheduleCommand()`：发送AI指令
- `executeScheduleCommand()`：执行日程操作指令

### API 配置

- **同步接口**：`https://open.bigmodel.cn/api/paas/v4/chat/completions`
- **支持模型**：GLM-4.7-Flash、GLM-4V-Flash、GLM-4.6V-Flash、GLM-4.1V-Thinking-Flash、GLM-4-Flash

## 注意事项

1. **API Key 安全**：请妥善保管您的智谱AI API Key，避免泄露
2. **数据备份**：建议定期备份日程数据，以防丢失
3. **权限说明**：脚本需要 `GM_addStyle`、`GM_setValue`、`GM_getValue` 权限
4. **兼容性**：支持主流现代浏览器，如 Chrome、Firefox、Edge
5. **性能优化**：脚本采用高效的渲染方式，即使在日程较多时也能保持流畅

## 故障排除

### 常见问题
1. **AI 执行失败**：检查 API Key 是否正确，网络连接是否正常
2. **日程不显示**：刷新页面或检查本地存储是否正常
3. **界面异常**：尝试重新安装脚本或清除浏览器缓存
